## Bash程序的创建和执行

### 1.创建

&emsp; 创建方式有两种：

- 直接通过文本编辑器进行编辑，然后将文件保存为`xxx.sh`的格式

- 通过命令行指令`nano xxx.sh`来在终端内进行编辑，进而保存到当前打开的位置

### 2.执行

&emsp; 执行方式也有两种：

- 使用`bash`指令：

  ```
    bash xxx.sh
  ```

- 使用`chmod a+x`指令修改文件权限：
  
  ```
    chmod a+x xxx.sh
    ./xxx..sh
  ```
